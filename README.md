## 🛠️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/upendrakda/Portfolio.git

cd Portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

---

## 📦 Build for production

```bash
npm run build
```

---

## 🚀 Deployment

This project is deployed using GitHub Pages.

```bash
npm run deploy
```
